# 化学和离子方程式配平器

一个用C++编写的命令行应用程序，用于自动配平化学和离子方程式。该应用程序包括强大的错误处理机制，防止输入格式错误，并附带测试用例确保功能正确。

## 功能

- **自动配平**：能够配平化学和离子方程式。
- **错误检测**：识别并提示用户输入格式错误。
- **单元测试**：包含测试用例，确保配平功能的正确性。

## 前提条件

- **C++ 编译器**：支持C++11标准（例如 `g++`）。
- **CMake**：用于构建项目。
